# Online Examination System

A web-based Online Examination System built using Django that helps organizations conduct online assessments for freshers and candidates. The platform allows administrators to manage exams, question banks, candidates, and results efficiently.

## 🚀 Features

- User Authentication & Authorization
- Admin Dashboard
- Candidate Registration & Login
- Question Bank Management
- Online Exam Interface
- Automatic Score Calculation
- Result Generation & Management
- Responsive User Interface
- Secure Database Storage

## 🛠️ Tech Stack

### Backend
- Python
- Django

### Frontend
- HTML5
- CSS3
- JavaScript

### Database
- SQLite (Default Django Database)

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/anujkus45/online-examination-system.git
```

### Navigate to Project

```bash
cd online-examination-system
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Create Superuser

```bash
python manage.py createsuperuser
```

### Run Server

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
